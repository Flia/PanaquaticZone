# Unified Panaquatic Growing Zone

A Rimworld mod which adds a standalone aquatic growing zone. It takes plants from mods like Vanilla Plants Expanded - More Plants, [LTS]Biosphere, and Goji's Fantasy Race: Merren, and puts them into a unified gizmo. As its name implies, it also allows growing in ocean tiles and define salinity preference for plants.
